### For Production Use:

1. **Instruments:**
   - Source high-quality photos (minimum 800x600, preferably higher)
   - Show the instrument clearly, well-lit, against neutral background
   - Include photos of instruments being played where possible
   - Consider adding 360Â° views for key instruments

2. **Artists:**
   - Use official press photos or properly licensed images
   - Portrait orientation (4:5 ratio recommended)
   - Professional headshots for profile images
   - Action shots for performance images
   - Ensure proper copyright/licensing

3. **Performance:**
   - Capture live performance scenes
   - Show regional costumes, settings, and contexts
   - Wide shots that show cultural environment
   - Landscape orientation (3:2 ratio)

4. **Image Optimization:**
   - Save as JPEG or WebP format
   - Compress for web (use tools like ImageOptim, TinyPNG)
   - Keep file sizes under 200KB for fast loading
   - Create responsive variants (@1x, @2x for retina displays)

5. **File Naming:**
   - Keep the same filenames to avoid breaking references
   - Use lowercase, hyphenated names
   - No spaces or special characters
