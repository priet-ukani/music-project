# India Map Image

Please save the India map image as `india-map.png` in this directory.

The image should be:
- Size: 882x1024 pixels (or similar aspect ratio)
- Format: PNG
- Content: Colored map of India with state boundaries

You can:
1. Download from: https://www.mapsofindia.com/maps/india/india-political-map.htm
2. Or use the reference image provided
3. Save it as `/home/kevin0099/music-project/website/public/images/india-map.png`

The component will overlay semi-transparent colored regions on top of this image
to make states clickable.
