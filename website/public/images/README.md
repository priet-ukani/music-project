## Image Specifications

- **Format**: JPG or PNG
- **Dimensions**: Minimum 800x600px for instruments, 1200x800px for performance images
- **File Size**: Optimized for web (<500KB per image)
- **Aspect Ratio**: 4:3 or 16:9 preferred

## Sourcing Images

Images can be sourced from:
1. Public domain collections (Wikimedia Commons)
2. Creative Commons licensed photos
3. Original photography with proper permissions
4. Government cultural archives
5. Museum collections with appropriate licensing
